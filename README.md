# JavaWeb_course

#### 介绍

Java技术栈之Javaweb学习

#### 概述

用Java技术来解决相关web互联网领域的技术栈，Internet上供外界访问的Web资源分为：静态web资源（如html 页面）：指web页面中供人们浏览的数据始终是不变。动态web资源：指web页面中供人们浏览的数据是由程序产生的，不同时间点访问web页面看到的内容各不相同。
